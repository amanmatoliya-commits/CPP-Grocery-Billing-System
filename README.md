# Apna Grocery Store Billing System

A console-based Grocery Store Billing System developed in C++ using Object-Oriented Programming concepts.

## Features

- Product Management
- Quantity and Unit Support
- Automatic Billing
- Discount Calculation
- Date and Time Generation
- Bill History Storage
- File Handling

## Technologies

- C++
- OOP Concepts
- File Handling
- STL

## Sample Products

- Sugar
- Milk
- Tea
- Coffee
- Tata Salt
- Refined Oil
- Maida
- Eggs
